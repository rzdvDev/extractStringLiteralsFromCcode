C++ project for SoftQA.

Описание
--------

Программа для извлечения строковых литералов из C-кода и записи результатов в файл.

Что реализовано
-----------------------

- Парсер (`include/parser.h`, `src/parser.cpp`): обход текста по строкам, поиск строковых литералов и формирование списка `Span` с позициями в тексте.
- Типы ошибок и их обработка (`AppError`) с понятными фабричными методами и выводом сообщений об ошибке.
- Ввод/вывод файлов (`include/file_io.h`, `src/file_io.cpp`): чтение входного файла построчно и запись результатов в выходной файл, сбор и возвращение ошибок через `std::vector<AppError>`.
- Точка входа `src/main.cpp`: консольный интерфейс, принимает два аргумента — путь к входному и выходному файлам. На Windows производится установка кодовой страницы консоли (CP1251).
- Система сборки CMake: targets `core` (библиотека), `app` (исполняемый файл), `tests` (unit-тесты). Добавлена опциональная цель `docs` для генерации документации Doxygen.
- Тесты (GoogleTest) находятся в каталоге `tests/`.
